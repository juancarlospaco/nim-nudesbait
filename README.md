# NudesBait

- Make **Fake** *"Nudes"* folders using tiny Wikipedia images (3 per folder) on [NimScript](https://nim-lang.github.io/Nim/nims.html) to *"Bait"* annoying harrasing people

[<img src="https://img.youtube.com/vi/DDBB34kH7gw/maxresdefault.jpg" width="99%">](https://youtu.be/DDBB34kH7gw "4 Minute Video to understand whats this!")


# Use

```nim
nim nudesbait.nims
```

- Requires Internet connection and `curl` (wont use too much disk nor network).
- Disable Folder Thumbnails, leave PC alone for a minute, record the activity somehow. You can delete the script after.
- You can also ZIP it and send it so someone asking to *Send Nudes*.


# Install

```console
curl -LO https://raw.githubusercontent.com/juancarlospaco/nim-nudesbait/master/nudesbait.nims
```


# Uninstall

```console
rm nudesbait.nims
```
